2 职业目标（短/中/长期）
- 1年目标（大一结束）
  - 扎实完成建筑基础课程（建筑制图、建筑史等）
  - 熟练掌握 AutoCAD、SketchUp 基础操作，完成至少 2 个小型项目表达（手绘+模型+图纸）
  - 参加至少一次校内/校外建筑工作坊或竞赛
- 3年目标（大三）
  - 熟练使用 AutoCAD、SketchUp、Photoshop，开始掌握 Revit 或 Rhino
  - 在课程设计中担任组内组织者或负责人，形成 3–5 件可展示的作品

- 5年目标（毕业后 1–2 年）
  - 在设计院或建筑事务所担任建筑设计师或助理设计师
  - 能独立完成方案阶段的小型或中型项目（含设计表达与技术性沟通）
  - 形成成熟个人作品集并确立专业方向（如住宅/公共建筑/城市更新等）
- 10年愿景
  - 成为具有独立项目能力的资深设计师，具备项目统筹能力；或在学术/研究方向继续深造与教学

3 能力模型与知识路线
- 核心能力
  - 设计能力：设计思维、场地分析、空间组织、方案推演
  - 技术表达：手绘、建筑制图（施工图基础）
  - 数字建模：AutoCAD → SketchUp → Revit / Rhino → BIM 协作
  - 表现能力：Photoshop、Illustrator、渲染（V-Ray/Enscape）
  - 实践能力：模型制作、构造与材料理解、工程常识
- 软件学习建议（按年级循序渐进）
  - 大一：手绘基础、AutoCAD、SketchUp、Photoshop 入门
  - 大二：深化设计课程，开始 Revit 或 Rhino 学习
  - 大三：BIM 基础与参数化工具（Grasshopper）入门，渲染与表现提升
- 软技能
  - 沟通表达（口头陈述、图纸解读、书面说明）
  - 项目管理（时间管理、分工协作）
  - 英语阅读能力（专业文献与国际项目沟通）

- 作品来源建议

  - 暑期实习与调研项目（实践性）
  - 个人实验/兴趣项目（体现独立思考）
- 示例作品集目录（按时间）
