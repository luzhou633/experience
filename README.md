# 黄钰杰 — 生涯发展报告与展示

本仓库包含为武汉大学建筑学 25 级新生黄钰杰准备的生涯发展材料（可导出为 PDF 与 PPT）。

包含文件：
- docs/黄钰杰_生涯发展报告.md — 报告源文本（可复制到 Word/WPS 并导出为 PDF）
- slides/黄钰杰_生涯发展展示_slides.md — PPT 每页文本与演讲备注（可用 PowerPoint/WPS 制作）

如何生成 PDF 与 PPTX（建议步骤）
1. 报告 PDF：在 Word/WPS 中新建文档，页面设置为 A4，按 docs/黄钰杰_生涯发展报告.md 的结构排版，插入武汉大学校徽（文件名：whu_logo.png）于封面右上角，导出为 PDF。
2. PPTX：在 PowerPoint/WPS 中新建 16:9 演示文稿，按 slides/黄钰杰_生涯发展展示_slides.md 的每页内容填充幻灯片，在封面与致谢页插入校徽，然后保存为 .pptx。导出前请压缩图片以保证文件小于 50 MB。

武汉大学校徽获取建议：请从武汉大学官网或学校宣传部门获取官方标识文件用于学术展示。

若你希望我直接上传最终的 PDF 与 PPTX 二进制文件到此仓库，请回复并在仓库设置中邀请我的账号作为协作者或提供一个上传权限。当前我已将源文本推送为 markdown 以便你直接使用。
